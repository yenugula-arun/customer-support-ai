# customer-support-ai

A repository for an AI-assisted customer support system.

## Proposed folder structure

- `frontend/` — UI application (web/mobile)
- `backend/` — server-side services
  - `create-ticket/` — create/ingest support tickets
  - `process-ticket/` — classify/route/answer tickets
  - `shared/` — shared utilities (models, clients, schemas)
- `infrastructure/` — IaC (Terraform/CDK), container configs, etc.
- `docs/` — architecture docs, runbooks
- `knowledge-base/` — documentation and content used by the AI
